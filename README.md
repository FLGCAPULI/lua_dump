# lua script dump

this is just a repository where i throw random lua scripts. most of them are standalone executables. some of them might do something mildly interesting. others are completely irrelevant. 

there is no overarching theme. there is no standard coding convention here. read the code if you want to know what it does.

sample:
```
loadstring(game:HttpGet("https://raw.githubusercontent.com/FLGCAPULI/lua_dump/refs/heads/main/umt.lua"))()
```
